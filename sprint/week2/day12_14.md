# Days 12-14: Stabilization

## Goal for Final Days
Repeat the same pattern types until solving feels automatic.

**Stop condition**: Medium solved in 30-35 min with correct code + passing tests.

---

## Daily Structure

### Each Day (90 min)
1. **Warm-up** (10 min): Template from memory + yesterday's micro-drill
2. **Timed problem** (35 min): Random medium from problem bank
3. **Review + Postmortem** (15 min)
4. **Second problem OR plumbing** (30 min)

---
